# React Essentials - Components, JSX, Props, State & More

This repository is a **work-in-progress learning project** for the React Essentials section.  
I’m using this project to explore React fundamentals, including components, JSX, props, state, and other core concepts.

## Goals

- Practice building React components
- Understand JSX syntax and how it works
- Learn how to pass and use props effectively
- Manage component state
- Experiment with basic React features and patterns

## Current Status

- Project setup using Vite (`npm create vite@latest`)
- Learning and experimenting with components and JSX
- Features and exercises will be added as I progress through the section

## Notes

- Expect unfinished code and experiments
- This is a personal learning project; not meant to be production-ready
- Will update README as new concepts are explored

## How to Run

1. Clone this repo

   ```bash
   git clone https://github.com/LTruter/my-first-react-app.git

   ```

2. Install dependencies

   ```bash
   npm install

   ```

3. Start development server

   ```bash
   npm run dev

   ```

4. Open in browser

   Follow the URL shown in the terminal (usually http://localhost:5173/)
